添加 / 跳转 方法：

1. 直接在 a 标签上加上相关属性

<a tab-href="url" tab-text="标题" >文本</a>

2. 直接用addPageTabs()

addPageTabs("url", "标题") //这里要注意要在父级调用方法 window.parent.addPageTabs("url", "标题")

关闭方法：

1. 直接用delPageTabs("url"),关闭当前用delPageTabs("tab-this")

delPageTabs("url")关闭指定路径，或者关闭当前页window.parent.delPageTabs("tab-this")
//这里要注意要在父级调用方法 window.parent.delPageTabs("url")
